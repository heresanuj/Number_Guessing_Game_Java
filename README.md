## ğŸ® How the Game Works
- The program stores numbers from **1 to 10** in an array.
- The computer randomly selects one number from this array.
- The user is asked to guess the selected number.
- After each guess, the program provides hints:
  - **Too High** if the guess is greater than the selected number
  - **Too Low** if the guess is smaller than the selected number
- The game continues until the user guesses the correct number.
- The total number of attempts is displayed at the end.

The program also validates user input and does not allow numbers outside the given range.
